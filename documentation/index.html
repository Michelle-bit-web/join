Join

This project is a web-based task manager and similar to the Kanban system. With Join you can create, organize, and track all tasks for your project. Join was generated using Angular CLI version 19.2.12. The Firebase CDKs for authentication and the firestore database were used to handle all upcoming data.

Live demo: Try Join yourself!
